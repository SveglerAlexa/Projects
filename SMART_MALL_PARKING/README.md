\# Smart Parking Management Mobile Application



A smart parking management mobile application designed for large commercial centers.



\## Features

\- Designed a smart parking management mobile application for large commercial centers.

\- Developed the frontend using \*\*Flutter\*\* and \*\*Dart\*\*.

\- Implemented the backend with \*\*Spring Boot\*\* and integrated a relational database for data storage.

\- Enabled real-time communication between the mobile app and the server via \*\*REST APIs\*\*.

\- Modeled functionalities such as:

&nbsp; - Parking visualization

&nbsp; - Parking spot reservation

&nbsp; - Vehicle parking status tracking



\## Future Improvements

\- Integration of indoor navigation/guidance to help drivers locate available parking spaces.
